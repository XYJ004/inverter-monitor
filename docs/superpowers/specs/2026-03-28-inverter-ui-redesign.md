# 逆变器监控系统 - UI 重构设计文档

**日期**: 2026-03-28  
**版本**: v2.0  
**目标**: 工业监控(SCADA) + 商业仪表盘(Grafana/Kibana) 风格结合

---

## 1. 整体架构

### 1.1 技术栈
- **前端**: React 18 + TypeScript + Vite + Tailwind CSS
- **图表**: Recharts
- **动画**: Framer Motion
- **后端**: Python Flask (端口 3002)
- **前端端口**: 3000

### 1.2 页面结构
```
[仪表盘] [电池详情] [发电分析] [告警记录] [系统设置]
```

### 1.3 开发优先级
1. **Phase 1**: 仪表盘 + 电池详情
2. **Phase 2**: 发电分析 + 告警记录
3. **Phase 3**: 系统设置

## 4. 后端 API 扩展

### 4.1 新增接口
```
GET /api/battery/history     - 电池 SOC 历史数据 (24小时)
GET /api/battery/stats       - 电池统计数据 (今日充放电)
GET /api/battery/thresholds  - 获取告警阈值
POST /api/battery/thresholds - 设置告警阈值
```

### 4.2 数据格式
```json
{
  "soc_history": [
    {"time": "00:00", "soc": 45.2},
    {"time": "01:00", "soc": 42.1},
    ...
  ],
  "today_charge": 18.3,
  "today_discharge": 15.6,
  "thresholds": {
    "soc_low": 10,
    "soc_high": 95,
    "temp_max": 45
  }
}
```
